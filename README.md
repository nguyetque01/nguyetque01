## 👩‍💻 About me

Backend Developer with **2+ years of experience** building and supporting enterprise systems across ERP, logistics, document processing, approval workflows, reporting, and external integrations.

I focus on backend systems where **APIs are clear, data remains consistent, permissions are predictable, and workflows are reliable in production**. My primary experience is with Node.js and TypeScript, with hands-on work using **C#, ASP.NET Core/.NET 8, Entity Framework Core, SQL Server, PostgreSQL, Redis, Docker, and GitLab CI/CD**.

- 🔭 End-to-end delivery: requirements, technical design, APIs, databases, testing, deployment, and production support
- 🧩 Reusable backend architecture, code generation, engineering conventions, and OpenAPI documentation
- ⚙️ Background jobs, webhooks, retry, idempotency, structured logging, health checks, and failure recovery
- 🤖 Practical AI integration with Gmail ingestion, OCR/LLM extraction, human validation, and traceable processing
- 🤝 Code review, technical documentation, cross-team delivery, and developer onboarding
